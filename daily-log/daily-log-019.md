## 트러블슈팅 & 배운 점

- `white-space: nowrap` 적용 시 텍스트가 벗어나는 문제  
  → 버튼 width 또는 padding 기반으로 해결
- Figma Hug는 텍스트 기반 자동 계산  
  → CSS에서는 padding 기반 구현이 더 자연스럽다는 점 배움
- 이미지 비율 깨짐  
  → `width: 100%; height: auto;`로 해결
- Figma와 웹의 간격이 다르게 보임  
  → 실제 브라우저 기준으로 margin/padding 직접 조정 필요
